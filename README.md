# 📊 Data Science Salary Dashboard (Excel)

## Overview

The **Data Science Salary Dashboard** is an interactive Excel dashboard designed to analyze salary trends across Data Science roles. Users can dynamically filter the data by **Job Title**, **Country**, and **Employment Type** to explore salary information and gain insights into compensation across different markets.

This project showcases modern Excel techniques, including dynamic arrays, advanced lookup functions, and interactive dashboard design, demonstrating how Excel can be used as a powerful business intelligence tool.

---

## 🎯 Business Objective

Create an interactive dashboard that enables users to explore salary trends across Data Science roles by job title, country, and employment type, helping identify compensation patterns and compare opportunities across different markets.

## Key Excel Functions Used

| Function | Purpose |
|----------|---------|
| `FILTER()` | Dynamically filters records based on user selections |
| `UNIQUE()` | Creates distinct lists for drop-down menus |
| `SORT()` | Automatically sorts dynamic lists |
| `XLOOKUP()` | Retrieves salary statistics based on selected filters |
| `SUMPRODUCT()` | Performs conditional counting and aggregation |
| `COUNTIF()` | Counts matching records |
| `TEXT()` | Formats values such as extracting month names |
| `LET()` | Simplifies complex formulas by storing intermediate calculations |
| `SEARCH()` | Identifies text within cells for conditional filtering |
| `IFS()` | Returns different outputs based on multiple conditions |
